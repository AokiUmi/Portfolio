Human Interaction Motion Generation From Text

ShanghaiTech VRVC Center, RhythMo

Sep 2022 - Jan 2023
Undergraduate researcher, supervised by Prof. Lan Xu

I join in a research group under VRVC center called RhythMo, it is now also a digital technology Co. Our team proposed a novel approach for human interaction motion generation from text. There are many reasearches about human motion generation, such as CLIP, DDPM, but very few about human interaction generation, so we want to generate some interaction between two people in our daily life.

To achieve our goal, we invited many volunteers to participate in our data collection, including actors who recorded daily interactions, and many professional practitioners to record some technical movements such as street dance, tai chi, basketball, fencing, etc.

I took active part in the data collection and annotation. During the research, I was able to experience first-hand the research process, learn to use research equipment, manage data and perform data processing. It is my first time to participate in research so I may not have made a very big contribution to method improvement and design, but I still learned a lot about generation model such as CLIP, DDPM. All these methods are widely used in the field of generation nowadays.

Here are some photos about my experience in VRVC lab

In addition, I also made a little project: human motion generation, under the guidance of a PhD student, Han Liang. We proposed a novel approach for human motion generation from text. Our model takes as input a text description of a motion, and outputs a sequence of motion frames that represent the motion described in the text. To obtain a complete motion generation model, we combine the CLIP motion embedding decoder with an a prior model that generates possible CLIP motion embedding from a given text heading, and then this embedding is used to condition a diffusion decoder which produces a final motion.

You can learn more about this little project Human-motion generation from text.

Chrome Dino Game

ShanghaiTech University, Computer Architecture I (CS110)

Chrome Dino pixel game on Longan Nano development board.

In this project, we implemented a Chrome Dino Runner Game using the Longan Nano development board. We utilized PlatformIO to interact with the development board and support cross-language programming. For example, we implemented most of the framework by C, but some of the auxiliary functions are implemented by RISC-V assembly. The functions can be invoked bidirectionally.

The game we implemented features:

  • A simple startup menu and diffuculty selection page.
  • A score board.
  • Randomly generated cactus and pterosaurs acting like obstacles.
  • The Dino can jump or creep controlled by the 2 buttons.
  • Super mode in which the dino can hit everything without failing the game.

Human Motion Generation From Text

ShanghaiTech University, VRVC lab, RhythMo

Introduction

Motion generation from text is a recently emerging field of research. It involves the task of generating a sequence of frames representing a human motion from a text description. This can be used for applications such as virtual reality, animation, and video games. The task is challenging as it requires understanding of natural language, as well as the ability to generate realistic motion.

In this work, we combine these two approaches to solve the problem of text conditional motion generation. We first train a diffusion decoder to invert the CLIP encoder. Our inverter is non-deterministic and can generate multiple motions corresponding to a given motion embedding. The presence of the encoder and its approximate inverse (decoder) allows for the ability to go beyond text-to-motion translation.

A space efficient algorithm for LCSK

ShanghaiTech University, Algorithm Design And Analysis (CS240)

The LCS problem is the problem of finding the largest sequence common to all sequences in a set (usually only two sequences). It has many applications such as pattern matching, file comparison and so on. The article we reseached proposed two space efficient algorithms for LCSk and LCS≥k problems based on algorithms of O(kn) space complexity, if the size of given sequences are both n.

RV32I-RVC Bidirectional Translator

ShanghaiTech University, Computer Architecture I, (CS110)

It is a project about writing an translator that translates RISC-V instructions to RISC-V compressed instructions for the first part. And for second part, we implement a translator that converts 16-bit RISC-V Compressed (RVC) instructions to equivalent 32-bit RISC-V instructions.

Here is the website introducing this project in detail: First part of project, Second part of project

Basically, it is not a hard job to understand but it takes our a long time to implement it. For me, I wrote 1000 lines code to finish the first part of this project, it really a huge challenge for me to write and debug such long and complicated code. I even made a specific form about it, it really took efforts.

Though the process is very challenging, I learn a lot from this precious experience, writing a huge project with so many lines code and finally succeeded on my own.
